Abaixo você encontrará alguns comandos de terminal que vão te ajudar a expandir os testes para nosso webserv.

1. Funcionamento

Execute o comando abaixo no terminal, e webserv deve ser capaz de funcionar com um arquivo padrao pre-definido.

./webserv

Execute o comando abaixo no terminal, para usar o curl.

curl -v http://localhost:8080

Execute o comando abaixo no terminal, para "multiple hostname".

curl -v -H "Host:site1" http://localhost:8080

Execute o comando abaixo no terminal, para testar metodo POST e "limit client body" (modificar no .config o client_max_body_size).

curl -X POST -H "Content-Type: plain/text" --data "OI" http://localhost:8080/curl.py

Execute o comando abaixo no terminal, para testar metodo DELETE.

curl -X DELETE http://localhost:8080/upload/curlposted.txt

Execute o comando abaixo no terminal, para testar metodo GET.

curl -X GET http://localhost:8080/get.py

Execute o comando abaixo no terminal, para testar metodo desconhecido.

curl -X AFE http://localhost:8080/
