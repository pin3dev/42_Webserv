Abaixo você encontrará alguns comandos de terminal que vão te ajudar a expandir os testes para nosso webserv.

1. Funcionamento

Execute o comando abaixo no terminal, e webserv deve ser capaz de funcionar com um arquivo padrao pre-definido.

./webserv

Execute o comando abaixo no terminal, para verificar se o servidor esta como nao bloqueante, verifique se os requests foram respondidos.

ab -n 1000 -c 100 http://localhost:8080/

Execute o comando abaixo no terminal, para verificar se o servidor esta com client bounced, verifique se o 11o cliente foi rejeitado.

2. Bibliotecas & Funcoes

Pesquise no projeto a funcao abaixo, e verifique se a mesma nao esta sendo usada para rodar outro webserv

excve()

Pesquise no projeto a funcao abaixo, e verifique se a mesma esta sendo chamada apenas uma vez e antes de todas os writes/send ou read/recv.

poll()
